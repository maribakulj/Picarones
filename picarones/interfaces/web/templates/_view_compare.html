Comparer deux runs
Diff entre deux benchmarks JSON — régressions, améliorations, écart CER significatif.
01
Disponible via CLI
La comparaison n'a pas encore d'endpoint HTTP — utilise la ligne de commande.
picarones compare run_a.json run_b.json \
  --output comparaison.html \
  --threshold 0.005 \
  --label-a "Tesseract 5" \
  --label-b "Mistral OCR"
Le rapport HTML met en évidence les moteurs qui ont régressé ou amélioré au-delà du seuil (CER absolu, par défaut 0,5 pp).
