Robustesse aux dégradations
Test de tenue d'un moteur face aux dégradations d'image (bruit, flou, rotation, résolution, binarisation).
01
Disponible via CLI
L'analyse de robustesse n'a pas encore d'endpoint HTTP — utilise la ligne de commande.
picarones robustness \
  --corpus ./corpus/ \
  --engine tesseract \
  --degradations noise,blur,rotation,resolution \
  --max-docs 50 \
  --output-json robustness.json
Pour un aperçu sans calcul réel, utilise picarones robustness --demo — génère un rapport HTML factice avec données synthétiques.
